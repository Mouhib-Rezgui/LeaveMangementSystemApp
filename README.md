## About
This Leave Management System allows employees to submit leave requests and managers to approve or reject them. It leverages Firebase for authentication, Firestore for real-time database updates, and provides an intuitive Flutter interface.

---

## Features
- **User Authentication:** Employee and Manager login/signup via Firebase Authentication
- **Leave Requests:** Submit, edit, and cancel leave requests
- **Approval Workflow:** Managers can approve or reject requests
- **Leave History:** Employees can view past requests
- **Real-time Updates:** Changes reflected instantly using Firestore
- **Notifications:** Optional (via Firebase Cloud Messaging)
- **User-friendly UI:** Built with Flutter for cross-platform support
